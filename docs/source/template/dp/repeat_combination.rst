::

    重複組合 H(n, m)
    N 個不同種類的物品，各有 a[i] 個，求從中選 m 個的方法數
    n[0] + n[1] + ... + n[N - 1] = m 的非負整數解的個數
    H(n, m) = C(n + m - 1, m) = C(n + m - 2, m) + C(n + m - 2, m - 1)
