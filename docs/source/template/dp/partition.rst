程式碼
************************

::

    將 N 個相同物放入 M 個相同箱（允許空箱）的方法數
    dp[i][j] = 將 i 個相同球放入 j 個相同箱的方法數
    if i < j, dp[i][j] = dp[i][i]
    if i >= j, 每個箱子都放入一顆球 + 至少有一個箱子不放球
        dp[i][j] = dp[i - j][j] + dp[i][j-1]
